To run the flask application:
  - Run database migration script
    ```
    flask db upgrade
    ```
  - Run the application
    ```
    flask run
    ```
    The default port for the Flask application is set to 5100. So the application can be accessed at the below URL.
    ```
    http://127.0.0.1:5100/
    ```
